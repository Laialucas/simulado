# Simulador Detran

Projeto desenvolvido em PHP, MySQL, HTML, CSS e JavaScript com o objetivo de simular uma prova teórica do Detran.

## Funcionalidades

* Iniciar simulado
* Responder questões
* Navegar entre perguntas
* Visualizar resultado final
* Integração com banco de dados MySQL

## Tecnologias utilizadas

* PHP
* MySQL
* HTML5
* CSS3
* JavaScript
* XAMPP

## Como executar

### 1. Instalar o XAMPP

Iniciar:

* Apache
* MySQL 

### 2. Criar banco de dados

Abrir o phpMyAdmin:

```text
http://localhost/phpmyadmin
```

Criar banco:

```sql
simulado
```

Executar os scripts SQL para criar a tabela e inserir as perguntas.

### 3. Colocar o projeto na pasta htdocs

```text
C:\xampp\htdocs\simulado
```

### 4. Executar o projeto

Abrir no navegador:

```text
http://localhost/simulado/index.html
```
